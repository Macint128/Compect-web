Compect Web 문법 안내서 (업데이트)

1. 기본 구조

Set(basic){
    Set(head){
        {lang:en}            // HTML lang
        {title:"Page Title"} // 페이지 제목
        let styles = [       // CSS 스타일
            *{ box-sizing:border-box } // 모든 요소에 적용
        ]
    }
    Set(body){
        (div){ "Content here" } // 본문 내용
    }
}

2. 스타일(let styles) 사용

- let styles = [ ... ] → <style> ... </style> 로 변환

- *{ ... } → HTML의 모든 요소에 적용 (CSS 전체 선택자)

let styles = [
    *{
        box-sizing: border-box;
        margin: 0;
        padding: 0;
    }
]

- 다른 CSS 선택자도 그대로 사용 가능:

let styles = [
    h1{ color: red; }
    p{ font-size: 16px; }
]
